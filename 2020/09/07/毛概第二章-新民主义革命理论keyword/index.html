毛概第二章 新民主义革命理论keyword

发表于 2020-09-07 分类于 Learning
本文字数： 1.4k 阅读时长 ≈ 1 分钟

总路线

  1. 第一次提出总路线——1939年《中国革命和中国共产党》
  2. 新民主主义革命
    • 性质——资产阶级革命（由革命对象决定）
    • 意义——属于世界无产阶级革命
    • 目的——建立无产阶级，社会主义
    • 首要问题——分清敌友
    • 中国革命的中心问题/新民主主义理论的核心问题——无产阶级领导权
    • 基本问题——农民土地问题
  3. 完整表述总路线、加入官僚资本主义——1948《在晋绥干部会议上的讲话》
  4. 完整表述新民主主义革命理论的著作——《新民主主义论》
  5. 中国富强独立前提——推翻帝国主义的压迫
  6. 反对敌人
    • 北洋军阀——国共合作
    • 新军阀——土地革命
    • 日本帝国主义——抗日战争
    • 蒋介石反动政权——解放战争
  7. 动力
    • 基本动力——无产阶级
    • 最可靠同盟军——农民
    • 动力之一——城市小资产阶级、民族资产阶级
  8. 民族资产阶级——又联合又斗争
  9. 实现领导权
    • 关键——建立以工农联盟为基础的广泛统一战线
    • 基本条件——率领取胜利、物质福利和政治教育
      *基础——对于农民的领导
    • 基本策略——坚持独立性
    • 坚强支柱——发展革命武装力量
    • 根本保障——政党建设
  10. 新民主主义革命分两步走不能分两场革命
    • 一次革命论（性质直接无产）——混淆了区别——左倾
    • 二次革命论（先资本后无产）——混淆了联系——右倾

基本纲领

  1. 简单阐述纲领——1940年《新民主主义革论》
  2. 具体阐述纲领——1945年《论联合政府》
  3. 政治纲领——新民主主义的共和国
    • 核心内容和基本准则——人民当家作主
    • 国体——各革命阶级联合专政
    • 政体——民主集中制的人民代表大会制度
  4. 经济纲领
    • 主要内容——没收土地归农民
    • 应有之义——没收官僚资本
    • 极具特色——保护民族工商业
    • 新民主和社会主义的双重性质——没收官僚资本
  5. 文化纲领
    • 文化是民族的——反对帝国主义压迫
    • 科学的——反对迷信
    • 人民大众——为工农大众服务

道路

道路提出

  1. 初步提出农村包围城市——《中国的红色政权…》《井冈山…》《星星之火….》
  2. 明确指出农村包围城市、长期武装斗争——1938年11月六届六中全会
  3. 革命方法——群众路线
  4. 革命道路——武装夺取政权、农村包围城市

道路形成

  1. 道路形成必要性——内无民主、外无独立、农民占大多数
  2. 道路形成可能性
    • 根本原因——极端不平衡
    • 群众基础好
    • 革命形势——客观条件
    • 红军存在——坚强后盾
    • 党的领导——主观条件

道路内容

  1. 内容——土地革命、武装斗争、农村革命根据地建设
  2. 土地革命——基本内容
  3. 武装斗争——主要形式
  4. 农村革命根据地——战略阵地+依托

三大法宝

  1. 提出——《共产党人发刊词》+统一战线、武装斗争、党的建设
  2. 武器——统一战线、武装斗争
  3. 英勇战士——党的建设
  4. 《共产党人发刊词》伟大工程——建设全国范围的广大群众性的马克思主义政党

统一战线

  1. 必要性——阶级状况、两头大中间小、不平衡性
  2. 中间——农民、城市小资产阶级以及其他中间阶级
  3. 可能性——诸多矛盾交织
  4. 必须坚持原则——独立自主（一切+统一战线=王明）
  5. 四个统一战线——国民革命联合战线、工农民主统一战线、抗日民主统一、人民民主统一
  6. 统一战线基础——工人同农民、广大知识分子以及其他劳动者
  7. 非常重要——工人阶级和非劳动人民

武装斗争

  1. 特点和优点——武装斗争、以武装的革命反对武装的反革命

人民军队

  1. 根本原则、根本前提、核心——坚持绝对领导
  2. 基本前提、根本准则、出发点与归属、不败之地——全心全意为人民

党的建设

  1. 中国特色社会主义最大优势——党的建设
  2. 主要矛盾——无产阶级和非无产阶级
  3. 党的建设首位——思想建设（革命年代）、政治建设（新时代）
  4. 工作作风——理论实践相结合、人民群众紧密联系（根本）、自我批评、《论联合政府》
  5. 区别于其他政党显著标志——三大作风
