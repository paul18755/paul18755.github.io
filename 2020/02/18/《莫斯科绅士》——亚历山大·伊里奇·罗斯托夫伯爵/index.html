《莫斯科绅士》——亚历山大·伊里奇·罗斯托夫伯爵

发表于 2020-02-18 更新于 2020-07-21 分类于 Read
本文字数： 1.1k 阅读时长 ≈ 1 分钟

亚历山大·伊里奇·罗斯托夫 伯爵

1月21日读完，整本书唯一值得吐槽的点可能就是那种奇怪的比喻腔调了，可能是因为翻译的问题，其原本语言应该是十分诙谐的。但是翻译过来感觉就不是那么趣味了，就好像以前那种“噢，我亲爱的达令，这可真是”等等那种。
除去这些之外，其实一开始会有些看不下去，可能是因为对于绅士和莫斯科俄罗斯的历史并不理解，很多东西是挺难代入的。但一开始与尼娜的玩耍还是挺有意思的。但是中途的许多故事，例如三巨头的组成和经历之类的、和安娜的情感、与政府官员上课经历等等其实可以说我是处于一种昏昏欲睡的情况下去读的。但是越到后面Book4Book5的时候越有一种沉浸感，就像读东野圭吾的小说不想停下的感觉。
整本书的高潮部分我觉得是在Book5，当我看到中间隔了一两卷之后，又再次出现了尼娜给伯爵的万能钥匙时，突然有一种感动和欣喜的感觉，就像是突然回忆起了自己生活中以前一件小事或者说突然捡到了自己很久之前丢失的一样东西，虽提不上宝贵，但却弥足珍贵，虽然书中的内容前后只有几百页纸的间隔。不得不说的是作者真的好似创造了一个世界，读到尼娜这把钥匙的时候感觉自己就在那个世界里面栩栩如生。
当然觉得是Book5也是因为谈及了伯爵的整个“逃跑计划”，相比大半个辈子都在酒店里，暮年却像少年一样勇敢的出逃，这是让我印象深刻的一点。当然与其说其在暮年之时才决心逃出大都会酒店，前人民的软禁，倒不如说其自软禁开始就在越狱，努力让自己还是保有之前的绅士生活、成为酒店的领班等等生活中的许多时间，很明显不是一个囚禁狱中的被软禁者的悲观表现。他给索菲亚的忠告很有意思：假如你不去掌控形势，你就会被形势掌控。我会想真正的掌控是什么样的？ 如伯爵从巴黎回到了莫斯科，从旁人的观点来说，回不回莫斯科肯定得考虑情况，十月革命的政治环境会有所考虑，在这个思考背景下，那么旁人便会认为伯爵有勇气等等其他一系列道德品质上的赞美之词。可以看看书一开头的法庭审判，对于伯爵前往巴黎，认为是“迫于形势远走高飞”；对于伯爵回来，会询问其“是不是抱有武装起义的打算，支持革命还是反对革命”。反观伯爵的回答，我想念这里的气候。这是绅士情怀对于无谓的阶级斗争的抗击，更是掌控与被掌控的深刻表达。每个人都有能力掌控自己的生活，但当以一个旁人的角度来考虑别人的生活的时候，往往会发现别人总被生活所掌控，因为旁人看的太多。而反观为什么我们会对自己的生活形势有不可掌控感，我想可能是因为我们也曾站在一个旁人的角度来考虑或评价我们自己的生活，当我们看的太多，可能往往看不见我们真正想要的，如何掌握形势？我觉得这个问题适合“自问自答”！当一个掌控形势的绅士再好不过了。或者是夜空中最亮的星也说不定☺️
