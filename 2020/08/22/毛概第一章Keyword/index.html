毛泽东思想形成和发展

  1. 时代背景—战争年代
  2. 实践基础—革命和建设的成功实践
  3. 理论初步形成—农村包围城市、武装夺取政权；中国革命的新道路
  4. 思想成熟—抗战后期、新民主主义革命理论系统阐述；1945七大写入党章;《中国革命和中国共产党》
  5. 思想继续发展—解放战争时期和新中国成立后、与实际结合

毛泽东思想的主要内容和活的灵魂

  1. 三个基本方面——实事求是、群众路线、独立自主
  2. 精髓—实事求是
  3. 实事求是—根本观点、思想路线的核心、根本思想路线
  4. 群众路线—生命线、根本工作路线、传家宝、根本出发点、区别于其他政党的根本标志
  5. 独立自主—立足点、优良传统、重要原则、必然结论
  6. 马克思主义中国化—同中国实践结合、即坚持又发展马克思主义、同优秀传统和文化结合

毛泽东思想的历史地位

  1. 首次提出“马克思主义中国化”/首次提出毛泽东思想—1938年、六届六中全会、《论新阶段》
  2. 系统概括毛泽东思想——1905中共七大、刘少奇《关于修改党的章程的报告》、十一届六中全会《关于建国以来党的若干历史问题的决议》
  3. 毛泽东思想—第一次历史性飞跃、革命与建设
  4. 中国特色社会主义理论体系——和平发展年代、改革开放现代化建设、二次飞跃
